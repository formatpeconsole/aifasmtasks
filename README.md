# aifasmtasks
Задачи, написанные на ассемблере чисто из спортивного интереса

Сделано с использованием FASM (https://flatassembler.net/)

Как запускать:

 Первым делом нужно положить папку tasks из архива в корень FASM

Порядок расположения файлов:

![alt text](https://i.imgur.com/RJgXzJ2.png)

После этого запустить fasmw.exe, выбрать пункт File -> Open

Для запуска можно использовать несколько способов:
1) Использовать клавишу F9
2) Открыть контекстное меню Run -> Run

Если нужно запустить через консоль 
То алгоритм идет следующим образом:
1) Собираем

		cmd>fasm tasks/even_odd.ASM

3) Запускаем
   
		cmd>tasks\even_odd.EXE
